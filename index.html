Website Cloner for Termux

A powerful tool with download size display and confirmation

Features

  • Clone any public website to your Android device
  • Displays estimated download size
  • Saves files in ~/storage/downloads/
  • Names folder by domain
  • One-word usage: clone
  • Clean UI for Termux

Installation

Step 1: Install Termux

Download from F-Droid for best compatibility.

Step 2: One-line install command

curl -L -o install.sh https://raw.githubusercontent.com/Zayan-Coder/Website-cloner/main/install.sh && bash install.sh
